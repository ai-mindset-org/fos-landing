## 8. VISUAL PATTERNS -- mickel.tech Inspiration

### 8.1 Corner Bracket Card Decoration

Cards with ASCII corner brackets as decorative elements:

```
╔                                          ╗
  Card content here
  Title, description, etc.
╚                                          ╝
```

**Implementation:** CSS `::before` and `::after` pseudo-elements on cards. Use `content: "╔"` etc. positioned at corners. Color: dim cyan or border color. Creates a retro-terminal aesthetic without full borders.

### 8.2 SYSTEM STATUS Card Layout

Label-value pairs in a terminal-style status panel:

```
┌─ SYSTEM STATUS ──────────────────────────┐
│                                          │
│  MODEL        Claude Opus 4              │
│  CONTEXT      200K tokens                │
│  SKILLS       18 active                  │
│  EVALS        T:✓ R:✓ C:✓               │
│  STATUS       ● OPERATIONAL              │
│                                          │
└──────────────────────────────────────────┘
```

**Implementation:** Grid or table layout inside a terminal-styled card. Left column = label (dim color, uppercase, letter-spacing), right column = value (cyan or white). Monospace font throughout. Thin border with header label.

### 8.3 Benchmark Cards with Scores

### 8.5 Glitch Text Effect on Titles

CSS glitch animation on section titles and hero text:

```css
.glitch-text {
  position: relative;
}
.glitch-text::before,
.glitch-text::after {
  content: attr(data-text);
  position: absolute;
  left: 0; top: 0;
  width: 100%; height: 100%;
}
.glitch-text::before {
  /* red/cyan color shift */
  color: var(--cyan);
  clip-path: /* random horizontal slices */;
  animation: glitch-1 2s infinite linear alternate-reverse;
}
.glitch-text::after {
  color: var(--pink);
  clip-path: /* different slices */;
  animation: glitch-2 3s infinite linear alternate-reverse;
}
```

**Usage:** Apply sparingly -- hero title, section headers on hover, the "OPEN CHANNEL" title. Triggered on scroll-reveal or hover, not constant.

### 8.6 [ BRACKET ] Navigation and Buttons

Navigation links and buttons wrapped in square brackets:

```
[ about ]  [ program ]  [ speakers ]  [ console ]  [ pricing ]
```

Already partially implemented: `[о нас]`, `[программа]`, etc. Enhance with:
- Brackets as part of the text, not decoration
- On hover: brackets glow cyan
- CTA buttons: `[ APPLY NOW ]` style

### 8.7 Scanline Overlay on Terminals

CRT-style horizontal scanline overlay on terminal components:

```css
.scanline-overlay::after {
  content: '';
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
  background: repeating-linear-gradient(
    0deg,
    transparent,
    transparent 2px,
    rgba(0, 0, 0, 0.03) 2px,
    rgba(0, 0, 0, 0.03) 4px
  );
  pointer-events: none;
  z-index: 2;
}
```

**Usage:** Apply to all terminal components (.terminal, .interactive-terminal, .code-block). Very subtle opacity (0.03-0.05). Creates CRT monitor feel.

## 10. EVALS SECTION CONTENT

### Concept

"Evals are the new PRD" -- Hamel Husain

Before you automate -- you measure. Before you scale -- you evaluate. Every skill, every agent output, every process gets a binary pass/fail evaluation. No scales. No subjectivity. Ship or fix.

### Rubric

| Area | Key Criteria |
|------|-------------|
| **T (Text)** | Structure, clarity, insights, actionable |
| **R (Rules)** | Naming convention, folders, formatting |
| **C (Code)** | Works, minimalism, idiomatic |

### Terminal Display (for landing page)

```
$ skill "daily-brief" --run --eval

источники: calendar, gmail, telegram, linear
сканирую 47 непрочитанных...

-> 3 встречи сегодня (первая: 10:00 product sync)
-> 2 срочных треда в telegram
-> дедлайн спринта: пятница

бриф сохранён -> daily/2026-02-10.md

---
eval: T:✓ R:✓ C:○
  T: структура чёткая, actions конкретны
  R: naming корректен, папка верна
  C: n/a (текстовый артефакт)
result: PASS
```

### Status Panel (for "System Status" card)

```

RUBRIC              T / R / C
MODE                Binary Pass/Fail
SKILLS EVALUATED    18
PASS RATE           94.4%
LAST EVAL           2m ago
```
